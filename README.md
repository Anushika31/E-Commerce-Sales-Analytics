# E-Commerce Sales & Customer Analytics Dashboard

## Project Overview

This project analyzes Brazilian E-Commerce sales data using Python, Power BI.

The objective is to identify sales trends, customer behavior, product performance and delivery efficiency.

## Tools Used

- Python
- Pandas
- Power BI
- DAX
- Data Modeling

## KPIs

- Total Revenue
- Total Orders
- Total Customers
- Average Order Value
- Repeat Customer Rate
- Average Rating
- Late Delivery %

## Key Insights

- Revenue increased by 20.0% in 2018 vs PY.
- Orders increased by 20.6% in 2028 vs PY.
- Health & Beauty generates highest revenue.
- 2018 generated the most revenue comparatively to previous years.
- Average customer rating is 4.02.
- Only 2.1% customers are repeat buyers which is relatively low.
- Late deliveries account for 8.6%.
- São Paulo contributes maximum sales.
